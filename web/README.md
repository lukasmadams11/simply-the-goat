# Simply the GOAT — Web

Marketing site for a wildfire-mitigation company that uses goat grazing.

Stack: React 18 + Vite + Tailwind CSS + react-router + lucide-react.

## Run

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production bundle into dist/
npm run preview  # serve the built bundle
```

## Brand tokens

Defined in `tailwind.config.js`.

- `bg-cream` `#FEFAE0` — page background
- `text-forest` `#1B4332` — headings, primary
- `text-ochre` `#BC6C25` — accents, active states
- `text-slate` `#2B2D42` — body

Headings use `font-serif` (Playfair Display). Body uses `font-sans` (Inter).
